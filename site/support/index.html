Need help with the app?

We use this page for App Store support information and production app help. If something is unclear, we want to make the next step obvious.

Best ways to get support

  • For app issues, feature requests, or bug reports, use the GitHub repository issue tracker.
  • Include your device model, iOS version, and a short description of what happened.
  • If the problem is related to an order, include the order number shown in the app.

Account help

  • You can sign out from the Profile tab inside the app.
  • You can delete your account from the Profile tab inside the app.
  • Deleting an account removes app access and stored contact details. Existing order records are retained in anonymized form for kitchen history and operational records.
